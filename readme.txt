To compile the code:

1. Copy the top-level files in the repo to: <Arduino-Sketch-Folder>/libraries/microgranny

2. Clone: https://github.com/anyway59/SdFat_bastlMicroGranny and copy contents to <Arduino-Sketch-Folder>/libraries/SDFat

3. open microGranny2_5 or other version from the examples folder of this repo using Arduino IDE, then select Arduino Uno and the correct serial port before uploading
